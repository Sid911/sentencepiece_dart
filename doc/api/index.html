Sentencepiece Dart

Sentencepiece Dart is a wrapper plugin for original Sentencepiece. C++ library. The plugin makes use of a little modified version of sentencepiece source which can be found in this fork.

Note : I made this plugin for my own use (so there might be some unimplemented things) but open sourcing the plugin means more people can use it even with little modification.
